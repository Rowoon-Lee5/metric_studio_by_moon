# 네 개의 2차 연구 질문

## 1. 알파는 연구 지연보다 오래 사는가

과거 60개월에서 가장 성과가 좋았던 신호·유동성 조합을 선택한 뒤, 0·1·3·6·12개월 늦게 실행한다. 지연 후 성과가 무너지면 알파의 문제가 아니라 연구 프로세스의 시간 문제가 된다.

## 2. 유동성 필터는 정보의 검열 장치인가

전체 시장에서 선택한 30종목과 유동성 필터 뒤 선택한 30종목의 교집합을 계산한다. 필터가 강할수록 선택 재현율과 신호 분산이 줄어든다면, 필터는 비용뿐 아니라 신호가 작동할 표본 공간도 제거한다.

## 3. 관심은 예측 가능성을 제거하는가

현재 데이터는 관심을 관측하지 못한다. 거래회전율을 대체변수로 두고, 회전율 5분위별 신호 Rank IC만 비교한다. 이 결과를 인기주의 인과효과로 해석하지 않는다.

## 4. 포트폴리오는 실패 시나리오의 집합인가

모멘텀 상위 60종목에서 고른 뒤, 과거 시장 하위 20% 달의 손실이 작고 변동성이 낮았던 30종목을 남긴다. 이는 수익률 극대화가 아니라 시장 실패 국면에서 같은 이유로 무너지는 종목을 줄이려는 롱온리 구성 규칙이다.
