# 1편 블로그 증거 이미지

`tools/build_blog_assets.py`가 커밋된 결과 파일에서 다시 만드는 SVG 이미지다.

네이버 블로그에 1편을 게시할 때 아래 순서로 업로드한다.

1. `01_robustness_map.png` — 실험 설계 설명 바로 뒤
2. `02_evidence_decision_boundary.png` — 현실성 검정 결과 바로 뒤
3. `03_observed_news_attention.png` — 2편의 뉴스량 검증 결과 뒤
4. `04_model_consensus_cumulative.png` — 3편의 합의 포트폴리오 결과 뒤
5. `05_model_failure_coherence.png` — 4편의 실패 공분산 결과 뒤
6. `06_book_excerpt_popularity.png` — 2편 도입부의 책 발췌
7. `07_book_excerpt_strategy.png` — 3편 도입부의 책 발췌
8. `08_book_excerpt_mixed_results.png` — 4편 도입부의 책 발췌
9. `09_book_excerpt_liquidity.png` — 1편 도입부의 책 발췌
10. `10_suspension_screened_smallcap.png` — 1편 거래정지 보정 결과 뒤

PNG는 네이버 블로그 업로드용이고, SVG는 확대·재편집을 위한 원본이다.

책 원문 파일에는 페이지 이미지나 쪽수가 없어서, 발췌 카드는 제공된 2장 텍스트를 짧게 인용해 만든 이미지다. 실제 책 페이지를 촬영해 제공받으면 이 카드는 해당 사진으로 교체한다.
