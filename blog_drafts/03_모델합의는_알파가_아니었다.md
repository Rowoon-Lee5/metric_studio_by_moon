# [문병로 교수의 메트릭 스튜디오 02] 2장 시장관찰 ③ 모델 합의는 알파가 아니었다

2장에서 저자는 패턴과 운용 전략을 함께 최적화해야 하며, 어느 곳에서 작동한 전략이 다른 곳에서 실패할 수 있다고 말한다. 이 문장을 읽고 나면 여러 전략이 동시에 고른 종목이 더 안전해 보인다. 여러 번의 검증을 통과한 종목처럼 보이기 때문이다.

연구자는 모델 하나를 믿기 어렵기 때문에 여러 모델을 만든다. 그러면 자연스럽게 여러 모델이 공통으로 고른 종목을 더 믿게 된다. 직관적으로는 합의가 불확실성을 줄인다.

이 직관을 확인하기 위해 반전, 모멘텀, 저변동성, 소형주 신호에 여러 유동성 컷과 보유 종목 수를 적용했다. 매월 각 종목이 몇 개의 모델에서 선택됐는지를 세고, 가장 많은 표를 받은 30종목을 매수했다. 이를 인식론적 안정성 포트폴리오라고 불렀다.

결과는 실패였다. 모델 합의 상위 종목 포트폴리오의 CAGR은 -12.9%였다. 모델이 흔들려도 남는 종목을 고르려 했지만, 남은 것은 알파가 아니었다.

이 결과는 모델 합의가 무의미하다는 뜻이 아니다. 내가 만든 모델들이 같은 가격·거래량·시가총액 패널을 보고 있었기 때문이다. 여러 모델의 표는 독립된 증거가 아니라 같은 데이터 구조를 다른 방식으로 반복한 표일 수 있다.

합의는 독립성 없이 늘어나면 확신이 아니라 공통 편향의 증폭기가 된다. 서로 다른 모델이 아니라 서로 다른 데이터 생성 과정을 가진 모델이 필요하다. 가격 신호, 공시 텍스트, 뉴스, 수급, 호가 데이터가 서로 다른 방향을 가리킬 때만 합의는 정보가 될 수 있다.

책이 말한 전략의 조건부 유효성은 여기서 더 불편한 질문으로 이어진다. 다음 질문은 모델을 몇 개 더 만들 것인가가 아니다. 내 모델들은 서로 얼마나 독립적인가다. 포트폴리오의 분산처럼, 연구자의 가설도 분산돼야 한다.
