# 문병로 교수의 메트릭 스튜디오 02 | 최종 전체 연구 패키지

이 폴더는 게시 원고만이 아니라, 그 원고의 주장과 수치를 재현하기 위한 연구 기록 전체를 묶은 최종본이다.

## 폴더 구성

- `00_게시_자료` : Word 원고, 네이버 업로드용 PNG, Markdown 원고, 게시 순서
- `01_실험코드` : 알파 위상·다중검정·뉴스 관심도·모델 합의·실패 공분산·거래정지·상장폐지·DART 감사 코드
- `02_검증결과` : 각 실험이 만든 CSV·JSON 결과와 워크포워드·시간분할 감사 결과
- `03_연구노트` : 질문 정의, 검증 논리, 한계와 반증 조건을 적은 연구 노트
- `04_외부데이터_명세` : 아직 필요한 호가·기업행사·증권마스터 데이터의 형식과 DART 공시 탐색의 사용 조건

## 재현 순서

1. `03_연구노트/RESEARCH_QUESTIONS.md`에서 각 글의 질문을 확인한다.
2. `01_실험코드`의 실험을 실행한다.
3. `02_검증결과`에서 결과가 원고의 수치와 일치하는지 대조한다.
4. 게시할 때는 `00_게시_자료`만 사용한다.

## 포함하지 않은 것

원시 가격·거래량 데이터, DART 공시 원문 큐, 외부 뉴스 원천 파일, API 키와 `.env`는 이 패키지에 넣지 않았다. 원시 데이터는 용량·라이선스·재배포 문제를 피하기 위해 원래 위치에서 관리하고, API 키는 실행 환경변수로만 제공한다.
