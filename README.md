# 메트릭 스튜디오 02 — 연구 기록

## 질문

유동성 상위 몇 퍼센트라는 고정된 컷오프는 전략의 실현 가능성을 설명하는가, 아니면 운용 규모와 주문 충격에 따라 달라지는 용량 제약을 가리고 있는가?

## 구현

`experiments/metric_studio_02_capacity.py`는 지정된 원시 수정주가·거래량 엑셀 파일을 직접 스트리밍한다. 가격과 거래량의 열 순서가 일치하지 않아 종목코드로 재정렬한다. 월말마다 21일 평균 거래대금(ADV)을 계산하고, 유동성 상위 20~100%에서 직전 1개월 수익률 하위 30종목을 동일가중 매수해 다음 달 수익률을 측정한다.

체결비용 스트레스 테스트는 편도 `10bp + 50bp × √(종목당 주문금액 / 21일 ADV)`다. 이는 실제 체결비용 추정이 아니라, 유동성과 주문 크기의 비선형 관계를 명시한 가정이다.

## 첫 결과

표본은 1999-12-28~2026-05-04, 월별 318개 시점, 3,717개 종목이다. 이 반전 신호에서는 유동성 상위 20%의 총수익 CAGR이 -12.93%, 상위 80%는 -2.38%, 전체 유니버스는 4.08%였다. 1억원 기준 비용 차감 CAGR은 각각 -15.10%, -4.82%, 1.49%였다.

이 결과는 “80%가 최적”이라는 일반 법칙을 지지하지 않는다. 다만 이 반전 신호에서 유동성 필터가 비용을 줄이는 동시에 신호의 표본 공간을 훼손했다는 사실은 보여 준다.

## 확장 검증 묶음

`experiments/metric_studio_02_research_suite.py`는 시가총액 원시 파일까지 결합해 가격 수준, 규모, 거래회전율 프록시, 6개월 모멘텀, 저변동성, 충격 지속성, 손절 반사실, 현금 기반 롱온리 위험 오버레이를 같은 표본에서 검증한다. 질문의 정의와 식별 한계는 [RESEARCH_QUESTIONS.md](RESEARCH_QUESTIONS.md)에 기록했다.

첫 실행에서 가격 하위 20%-상위 20%의 다음 달 평균수익률 스프레드는 -2.42%p, 시총 스프레드는 -2.51%p, 거래회전율 프록시 스프레드는 -2.33%p였다. 6개월 모멘텀 상위-하위 20%의 이후 12개월 스프레드는 +3.13%p였다. 이 수치는 거래비용을 반영하지 않은 연관성이지 매매 권고가 아니다.

현금 변동성 타기(목표 연변동성 12%)는 동일가중 시장 포트폴리오의 MDD를 -59.0%에서 -34.0%로 낮췄지만 CAGR도 11.6%에서 7.3%로 낮췄다. 이것이 롱온리 헤지의 본질이다. 수익률을 보존하는 장치가 아니라 위험 예산을 줄이기 위해 기회비용을 지불하는 장치다.

## 2차 검증: 신호보다 연구 과정이 먼저 실패할 수 있다

`experiments/second_order_research.py`는 네 가지 질문을 실행한다. 60개월 과거 성과로 신호·유동성 조합을 고른 뒤 지연 실행하는 알파 반감기 실험, 유동성 컷이 전체 시장 선택 종목을 얼마나 제거하는지 보는 선택 재현율 실험, 거래회전율 분위별 예측 가능성 비교, 과거 위기월 손실과 변동성을 이용한 실패 인지형 롱온리 포트폴리오다.

60개월 롤링으로 사후 최고 전략을 선택한 실험은 실행 지연 0~12개월 모두 음의 CAGR이었다. 이 결과는 지연이 알파를 죽였다는 증거가 아니다. 오히려 선택 가능한 후보군이 부실하면 신속한 구현도 알파를 만들지 못한다는 반증이다. 6개월 모멘텀의 12개월 Rank IC가 거래회전율 하위 20%에서 0.044, 상위 20%에서 0.017로 낮아진 점은 관심 프록시가 높을수록 예측 가능성이 줄어든다는 가설과 일치하지만, 관심 데이터를 관측하지 못했으므로 인과 주장은 금지한다.

## 실제 뉴스량 반증

`experiments/news_attention_validation.py`는 외부 뉴스 패널의 월별 기사 수를 실제 관심도 변수로 사용한다. 70종목, 2000-06~2025-04의 8,399개 결합 관측치에서 뉴스량 하위·중간·상위 3분위의 6개월 모멘텀 이후 12개월 Rank IC는 각각 0.057, 0.040, 0.080이었다. 거래회전율 프록시에서 보였던 “관심이 높을수록 예측력이 사라진다”는 패턴은 재현되지 않았다. 따라서 거래회전율을 관심도의 대체변수로 취급했던 기존 해석은 기각한다.
